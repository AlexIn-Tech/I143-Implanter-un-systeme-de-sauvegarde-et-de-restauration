# Acteurs majeurs des sauvegardes cloud

- AWS , Microsoft Azure, Google Cloud, Blackbaze, etc
- Le cloud est souvent «facile» à utiliser, MAIS :
- Attention aux coûts ! ⚠️
  - Ingress (Entrée) : Le transfert des données vers le cloud
    - Généralement Gratuit
  - Egress (Sortie) : Transfert de données depuis le cloud vers un autre système.
    - Souvent facturé par volume de données transféré
- Reste intéressant pour 3-2-1 pour une PME.

---

# Outils gratuits / Open Source

- Rsync
- Description : Outil de synchronisation et de copie de fichiers localement ou à distance.
- Avantages :
  - Rapide et fiable pour les sauvegardes incrémentales.
  - Très flexible avec de nombreuses options en ligne de commande.
- Cas d’utilisation :
  - Sauvegarde locale ou sur un serveur distant via SSH.

---

# Outils gratuits / Open Source

- rClone
- Description : Gestion et synchronisation des fichiers avec des services de stockage cloud.
- Avantages :
  - Compatible avec plus de 50 fournisseurs cloud (AWS, Google Drive, etc.).
  - Prise en charge du chiffrement et de la compression.
- Cas d’utilisation :
  - Sauvegarde cloud et migration de données entre clouds.

---

# Outils gratuits / Open Source

- Duplicati
- Description : Solution de sauvegarde open-source avec une interface utilisateur.
- Avantages :
  - Chiffrement AES-256 intégré.
  - Planification et sauvegarde vers le cloud (Dropbox, Google Drive, etc.).
- Cas d’utilisation :
  - Sauvegardes locales et cloud pour utilisateurs non techniques.
